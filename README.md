# O2 Series Download
Download movies from O2TvSeries.com with just one click.

## How to Use
1. Clone repository
2. Run `yarn install` or `npm install` in the root directory to install dependencies
3. Finally use `yarn start` or `npm start` to run local build
4. Happy hacking
